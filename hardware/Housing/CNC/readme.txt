These files are CNC programs to cut (hog) a case from a 0.735" wood or MDF block. 
variation_2_ is for Adafruits 1.8" lcd with the SD card socket removed, the version 3 is for the same 1.8 tft
but on a larger carrier pcb from China. Use 1/8 dia endmill router bit.

The main pcb with the rotary encoder is the AIR_LCDuino board -- found on OSH PCB and documented in detail
at https://github.com/garyStofer/AIR_LCDuino.
